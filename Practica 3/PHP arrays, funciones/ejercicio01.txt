Ambos códigos son equivalentes ya que ambos realizan la misma acción de crear un arreglo $a con valores específicos.
Una manera fácil de verlo es utilizando json_encode para mostrar ambos arreglos:

Código 1:
{
  "0": 4,
  "color": "rojo",
  "sabor": "dulce",
  "forma": "redonda",
  "nombre": "manzana"
}

Código 2:
{
  "0": 4,
  "color": "rojo",
  "sabor": "dulce",
  "forma": "redonda",
  "nombre": "manzana"
}

Como podemos observar ambos resultados son iguales.
